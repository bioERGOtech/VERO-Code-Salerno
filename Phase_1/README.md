# VERO Code (Phase 1)

Reproducible notebooks for data profiling, cleaning, unit standardization, imputation, and visuals.

## Structure
- notebooks/: VERO_01 ... VERO_06
- data/: raw, interim, processed
- outputs/: profiling, imputation, standardization, visuals
- reports/: Phase1_Report.md

## Environment
- Install with pip: `pip install -r requirements.txt`
- Or conda: `conda env create -f environment.yml && conda activate vero`
